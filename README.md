octodex-ios
===========

A tiny app to display different octocats from the [octodex](https://octodex.github.com). You can swipe left, right, or shake to get a new octocat. It could be a lot cooler.

Running
=======

    $ bundle install --path vendor/gems
    $ bundle exec rake
